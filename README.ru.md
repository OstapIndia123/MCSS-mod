# MCSS

[![EN](https://img.shields.io/badge/README-English-blue)](README.md)

Простой мод для Minecraft, который связывает мир с внешним сервисом через WebSocket.
Идея в двух словах: ставите блок "Hub", мод подключается к вашему backend и
получает команды/отдает данные для вашей системы безопасности.

## Что делает мод

- подключает Minecraft к внешнему сервису по WebSocket;
- добавляет блоки: Hub, Hub Extension и Reader;
- хранит стабильный идентификатор хаба (`hubId`) для связи с backend;
- дает простую конфигурацию через файл `config/hubmod.yml`.

## Совместимость

- Minecraft: 1.21.11
- Fabric Loader: >= 0.18.2
- Fabric API: любая актуальная версия

## Установка

1) Установите Fabric Loader и Fabric API.
2) Скопируйте `.jar` мода в папку `mods`.
3) Запустите игру/сервер — конфиги появятся автоматически.

## Быстрый старт

1) Запустите мир один раз, чтобы создались конфиги.
2) Откройте `config/hubmod.yml` и укажите адрес вашего WebSocket-сервера.
3) Перезапустите игру/сервер.
4) В креативе откройте вкладку предметов "Hub" и поставьте блоки.

## Конфигурация

Файл: `config/hubmod.yml`

```yaml
# Включить дополнительный лог
debug: false

# Адрес WebSocket сервера
wsUrl: "ws://127.0.0.1:8080"

# Интервалы тестов соединения (в миллисекундах)
testPeriodMs: 300000
testFailAfterMs: 300000
```

Файл: `config/hubmod.json`

- содержит `hubId`, который создается автоматически при первом запуске;
- этот идентификатор используется для связи вашего мира с backend.

## Блоки

- **Hub** — основной блок, через него мир подключается к backend.
- **Hub Extension** — дополнительный блок для расширения хаба.
- **Reader** — блок-считыватель для взаимодействия с системой.
